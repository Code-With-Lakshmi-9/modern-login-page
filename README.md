# 🔐 Modern Login Form

A simple and stylish login form built using HTML & CSS with a modern glassmorphism design.

---

## ✨ Features

- ✅ Responsive design (mobile + desktop)
- ✅ Glassmorphism UI (blur + transparency)
- ✅ Clean and minimal layout
- ✅ Centered login box
- ✅ Hover effects on button
- ✅ Background image support

---

## 🛠️ Technologies Used

- HTML5
- CSS3
